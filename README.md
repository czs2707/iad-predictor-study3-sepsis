# IAD Risk Predictor - Study 3: Sepsis Patients

ML-based risk prediction tool for Incontinence-Associated Dermatitis (IAD) in sepsis patients.

## Access the Tool

**[Launch Predictor](https://czs2707.github.io/iad-predictor-study3-sepsis/)**

## About

- **Population**: Sepsis patients (all ages)
- **Model**: Logistic Regression
- **AUC**: 0.818
- **Variables**: 28 clinical parameters
- **IAD Prevalence**: 28.7%

This tool is for research and educational purposes only.
